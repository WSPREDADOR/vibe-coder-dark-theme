# Vibe Coder Dark Theme

A vibrant cyberpunk dark theme with neon colors inspired by Vibe Coder - perfect for developers who love bright, energetic coding environments.

## Features

- 🌟 **Neon Colors**: Vibrant green, pink, blue, and yellow accents
- 🌙 **Dark Background**: Deep cyberpunk-style dark theme
- 💻 **Syntax Highlighting**: Optimized for multiple programming languages
- 🎨 **UI Elements**: Consistent neon styling across all VS Code components

## Installation

### Option 1: From VS Code Marketplace (Recommended)
Search for "Vibe Coder Dark" in the VS Code Extensions marketplace.

### Option 2: Manual Installation
1. Download the `.vsix` file from the releases
2. In VS Code: `Extensions → Install from VSIX...`
3. Select the downloaded file

### Option 3: From Source
1. Clone this repository:
   ```bash
   git clone https://github.com/WSPREDADOR/vibe-coder-dark-theme.git
   ```
2. Open the folder in VS Code
3. Press `F5` to run in Extension Development Host
4. Package with `vsce package` for distribution

## Color Palette

- **Background**: Deep dark (#0a0a0a)
- **Primary Green**: Bright neon (#00ff88)
- **Accent Pink**: Vibrant magenta (#ff0080)
- **Secondary Blue**: Electric cyan (#00ccff)
- **Highlight Yellow**: Bright gold (#ffff00)
